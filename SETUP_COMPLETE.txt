🔐 SECURITY IMPLEMENTATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Authentication Flow:
   User Login → AuthService → API → Get Token → Store in localStorage
                                       ↓
   All Requests → AuthInterceptor adds Authorization header
                                       ↓
   Response Success? → Return data to component
   Response 401? → AuthInterceptor refreshes token → Retry request

Token Management:
   ✓ Access tokens stored in localStorage
   ✓ Refresh tokens stored in localStorage
   ✓ Automatic refresh 5 minutes before expiry
   ✓ Secure 401 error handling with retry logic
   ✓ Automatic logout on refresh failure

Route Protection:
   /auth/*        → NoAuthGuard (only for logged-out users)
   /dashboard/*   → AuthGuard (only for logged-in users)
   /             → Auto-redirect based on auth state
